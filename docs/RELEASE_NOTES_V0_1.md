# Release Notes: v0.1.0

## Definition of v0.1

v0.1 is a hardened MVP for bounded local research runs with deterministic artifact generation and inspectable outputs through CLI and API.

## Included

- Objective-to-report local workflow execution.
- Local artifact persistence with manifest + report + final result artifacts.
- CLI commands for run execution and run/artifact inspection.
- API endpoints for health, readiness, run creation, run retrieval, run listing, and artifact listing.
- Deterministic tests for core workflow, persistence, extraction fixtures, and interface paths.

## Deferred

- Browser automation implementation (bounded placeholder only).
- Advanced retrieval/analysis features outside MVP scope.
- Distributed storage and hosted infrastructure.
